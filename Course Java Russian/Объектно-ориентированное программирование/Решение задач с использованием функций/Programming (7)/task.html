+3 *5

Определить можно ли с использованием только операций «прибавить 3» и «умножить на 5» получить из числа 1 число ? (? - натуральное, не превышает 1 000 000). Разумеется, само число 1 получить можно, просто не применяя никаких операций.

Входные данные

Вводится число ?.

Выходные данные

Выведите слово YES, если число ? можно получить из числа 1, или NO - в противном случае.


Sample Input:
1

Sample Output:
YES


Sample Input:
27

Sample Output:
NO



Memory limit: 256 MB
Time limit: 8 seconds
