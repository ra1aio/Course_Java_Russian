Задача 8

Вводится натуральное число n (n > 1). На следующей строчке вводятся n целых чисел. Считайте их как массив на n элементов. Найдите количество элементов, которые больше предыдущего элемента.


Sample Input:
5
1 3 -4 6 2

Sample Output:
2


Sample Input:
4
5 3 2 0

Sample Output:
0



Memory limit: 256 MB
Time limit: 8 seconds
