Задача 5 (средне)

Вводится натуральное число n. На следующей строчке вводятся n целых чисел. Напишите код, который создаст массив с именем array на n элементов, в котором буду храниться введённые числа (элемент массива равен соответствующему числу в во второй строке). На следующей строке вводится 2 числа - индексы элементов, сумму которых вам надо найти. Выведите на экран сумму этих двух элементов. Если вдруг индекс индекс оказался вне пределов диапазона от 0 до n - 1, то выведите на экран слово Error. 


Sample Input:
5
12 3 -4 6 2
0 3

Sample Output:
18


Sample Input:
3
9 -4 3
1 10

Sample Output:
Error



Memory limit: 256 MB
Time limit: 8 seconds
