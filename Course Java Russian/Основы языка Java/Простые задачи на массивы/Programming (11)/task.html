Задача 12

Вводится натуральное число n. На следующей строчке вводятся n целых чисел. Считайте их как массив на n элементов. на следующей строке вводятся два индекса: l и r. Найдите среднее арифметическое всех элементов с индексами от l до r включительно. Выведите в ответ одно действительное число. Гарантируется, что 0 <= l <= r < n.


Sample Input:
5
1 2 3 2 3
1 2

Sample Output:
2.5


Sample Input:
8
1 2 3 1 2 1 2 1
3 6

Sample Output:
1.5



Memory limit: 256 MB
Time limit: 8 seconds
