Две равные строки

С клавиатуры построчно вводятся три строки. Если среди них есть ровно две равные строки то вывести "Yes", в противном случае - "No".


Sample Input:
String
Hello!
String

Sample Output:
Yes


Sample Input:
hello
String
Not String

Sample Output:
No
