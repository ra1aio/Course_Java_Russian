JComboBox

Создайте приложение, содержащее в себе элемент JComboBox. 

Ваше приложение должно содержать выпадающий список JComboBox, в котором будут хранится названия ваших любимых цветов. Ниже должна находится кнопка "Ответить", при нажатии на которую в надписи "Ответ" должен появляться выбранный вами цвет. Если же вы хотите ввести свой цвет, то для этого должен существовать отдельный ввод, состоящий из JCheckBox и JTextField. Если чекбокс не выбран, то у пользователя не должно быть возможности вводить текст в текстовое поле.

В качестве ответа пришлите ссылку на репозиторий с вашим проектом (конкретно на сайте github.com).
