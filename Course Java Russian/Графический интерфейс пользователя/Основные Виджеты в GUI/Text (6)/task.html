JTextArea

Создайте приложение, содержащее в себе элемент JTextArea. 

Ваше приложение должно содержать элемент JTextArea, в который будут попадать строки, которые пользователь будет вводить в JTextField, расположенный выше, по одной за раз. Между ними должна находится кнопка "Записать", при нажатии на которую cтрока из JTextField будет дописываться последней в JTextArea. Добавьте к вашему JTextArea элемент JScrollPane, который добавит ползунки, если вдруг текста в вашем JTextArea станет слишком много.

В качестве ответа пришлите ссылку на репозиторий с вашим проектом (конкретно на сайте github.com).
