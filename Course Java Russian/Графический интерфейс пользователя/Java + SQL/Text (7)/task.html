Много котиков

Напишите функцию add_more_cats(int n), которая будет добавлять в базу данных n случайный кошек.

Для создания случайного котика пользуйтесь функцией Math.random, которая генерирует случайное число от 0 до 1. Для генерации случайной породы используйте массив из слайда 5 или уже имеющуюся БД. Случайные имена можно получить из массива с кличками кошек. 

После добавьте в вашу базу данных 5 тысяч котиков.

В качестве ответа пришлите ссылку на репозиторий с вашим проектом (конкретно на сайте github.com).
