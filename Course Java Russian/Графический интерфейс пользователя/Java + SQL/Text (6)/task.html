Первый INSERT

Добавьте к вашему проекту функцию insert_cat(String name, String type, int age, Double weight), которая добавляет в базу данных кошку с переданными параметрами. Обратите внимание, что в качестве параметра type приходит сам тип кошки, а не его id. Соответственно, если такой тип уже есть в БД, то вам необходимо достать его id и добавить его в соответствующее поле. Если же такого типа ещё нет в нашей БД, то добавить этот тип в таблицу types, получить его id и только потом добавить нашего котика с новым id. Добавьте 3 каких-нибудь котиков в вашу БД. Хотя бы один из них должен иметь тип, которого ещё нет в вашей БД.

Проверьте после этого вашу БД с помощью SQLite.

В качестве ответа пришлите ссылку на репозиторий с вашим проектом (конкретно на сайте github.com).
