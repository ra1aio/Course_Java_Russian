Класс JOptionPane

Очень часто в разных программах возникают маленькие диалоговые окна: для выбора чего-либо, подтверждения ввода данных, для ввода имени и т.д. Такие окна как правило очень однообразны и не хочется писать для них целый отдельный класс со всей логикой. поэтому в java существует отдельный класс - JOptionPane, отвечающий за подобные окна.
