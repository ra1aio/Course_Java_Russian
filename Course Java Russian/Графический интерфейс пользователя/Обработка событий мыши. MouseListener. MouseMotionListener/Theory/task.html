Интерфейс MouseListener

Для обработки событий мыши в Java существует интерфейс MouseListener.

Он содержит 5 методов:

  • mouseClicked(MouseEvent e) - обрабатывает "клик" мышки (между нажатием и отпусканием клавиши координаты не менялись)
  • mousePressed(MouseEvent e) - обрабатывает нажатие на клавишу
  • mouseReleased(MouseEvent e) - обрабатывает отпускание клавиши
  • mouseEntered(MouseEvent e) - вызывается при попадании курсора мыши в область компонента
  • mouseExited(MouseEvent e) - вызывается при выходе курсора мыши из области компонента
