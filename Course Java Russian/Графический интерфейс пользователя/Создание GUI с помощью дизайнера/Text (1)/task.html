Текстовый редактор

Создайте элементарный текстовый редактор с возможностью загрузки данных из файла и записи в файл. Добавьте возможность "Сохранить как". Выбор файлов для открытие и сохранения реализуйте через диалоговые окна выбора файла.

В качестве ответа пришлите ссылку на репозиторий с вашим проектом (конкретно на сайте github.com).
