Земля

Найдите в интернете картинки солнца и Земли. Добавьте картинку солнца в центр вашего окна. Заставьте Землю крутиться вокруг неё по окружности. В качестве ответа пришлите ссылку на репозиторий с вашим проектом (конкретно на сайте github.com).

Для движения по окружности рекомендую использовать параметрическое уравнение окружности.
