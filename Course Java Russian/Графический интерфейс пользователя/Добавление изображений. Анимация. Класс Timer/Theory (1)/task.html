Класс Timer (java.swing)

Класс Timer из библиотеки Swing очень прост в работе. Для него достаточно просто указать количество миллисекунд и  ActionListener, который будет принимать событие. Посмотрим на код:

import javax.swing.*;
import java.awt.event.ActionEvent;
import java.awt.event.ActionListener;
import java.text.DateFormat;
import java.text.SimpleDateFormat;
import java.util.Date;
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        ActionListener messageAL = new ActionListener() {
            @Override
            public void actionPerformed(ActionEvent e) {
                DateFormat dateFormat = new SimpleDateFormat("yyyy/MM/dd HH:mm:ss");
                Date date = new Date();
                System.out.println("Новое Сообщение! Получено в " + dateFormat.format(date));
            }
        };
        Timer myTimer = new Timer(3000, messageAL);
        myTimer.start();
        Scanner sc = new Scanner(System.in);
        sc.nextLine();
    }
}

Сначала создаётся ActionListener, в котором есть маленькая функция, которая выводит на экран сообщение. Не будем останавливаться подробно на её коде, так как до его назначения легко догадаться. После создания обработчика мы создаём Timer, который будет вызывать событие каждые 3 секунды. Это событие будет обрабатывать на ActionListener. После того, как мы создали таймер, его нужно запустить.

Последние 2 строчки добавлены лишь для того, чтобы программа не завершалась. При работе с графикой их прописывать не надо, так как программа не завершится до тех пор, пока окно не закроется.
