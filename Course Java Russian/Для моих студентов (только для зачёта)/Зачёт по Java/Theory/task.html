Зачёт по Java

Зачёт включает в себя 5 задач на знание языка программирования java. 

Эти 5 задач суммарно дают 100 баллов. Обратите внимание, что вы можете получить за задачу неполный балл, если она проходит не все тесты.

Для получения зачёта нужно решить первые три задачи минимум на 50 баллов (каждая задача оценивается в 20 баллов максимум) и решить последние 2 задачи, которые проверяет преподаватель. Выполнение работы рассчитано примерно на 1,5 часа. Удачи!!!
